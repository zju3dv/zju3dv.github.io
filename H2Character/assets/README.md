# Assets Directory

This directory contains all media assets for the H2Char project website.

## Directory Structure

```
assets/
├── images/           # Image files
│   ├── teaser.jpg   # Main teaser image
│   ├── hero-bg.jpg  # Hero section background
│   ├── paper-figure.jpg # Paper figure
│   └── video-poster.jpg # Video thumbnail
└── videos/          # Video files
    ├── explanation.mp4 # 2-minute explanation video
    ├── walking.mp4     # Motion demonstration videos
    ├── running.mp4
    ├── jumping.mp4
    ├── dancing.mp4
    ├── fighting.mp4
    ├── idle.mp4
    ├── climbing.mp4
    ├── sitting.mp4
    ├── gesturing.mp4
    └── complex.mp4
```

## File Requirements

### Images
- **teaser.jpg**: Main project teaser image (recommended: 1200x800px, JPEG/PNG)
- **hero-bg.jpg**: Hero section background (recommended: 1920x1080px, high quality)
- **paper-figure.jpg**: Paper figure/image (recommended: 800x600px)
- **video-poster.jpg**: Video thumbnail/placeholder (recommended: 640x360px)

### Videos
- **explanation.mp4**: 2-minute project explanation video (max 200MB)
- Motion demonstration videos: 10-30 seconds each, showing different motions

## Adding Your Assets

1. **Replace placeholder images**:
   - Take screenshots of your project results
   - Create professional-looking teaser images
   - Ensure images are properly compressed

2. **Add your videos**:
   - Record motion demonstrations
   - Create the 2-minute explanation video
   - Compress videos for web delivery

3. **Optimize for web**:
   - Use appropriate compression
   - Keep file sizes reasonable
   - Test loading times

## File Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ `character-walking.mp4`
- ❌ `Character_Walking.MP4`
